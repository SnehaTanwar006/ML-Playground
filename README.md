## 📌 Overview

This repository serves as a structured reference for ML algorithm implementations. Instead of scattered individual repos, everything is indexed here — organized by learning paradigm.

---

## 📂 Repository Structure

```
ML-Playground/
├── supervised/
│   ├── linear-regression/      → Linear Regression (OLS, gradient descent)
│   ├── logistic-regression/    → Logistic Regression (binary classification)
│   └── neural-networks/        → Artificial Neural Networks (ANN)
├── unsupervised/
│   └── k-means/                → K-Means Clustering analysis
├── deep_learning/
│   ├── cnn-mnist/              → CNN for MNIST digit classification
│   └── cnn-augmentation/       → CNN with data augmentation (Fish recognition)
└── README.md
```

## 🚀 Getting Started

```bash
# Clone this index repo
git clone https://github.com/SnehaTanwar006/ML-Playground.git
cd ML-Playground

# Install common dependencies
pip install numpy pandas scikit-learn tensorflow matplotlib seaborn jupyter

# Launch Jupyter
jupyter notebook
```

Then navigate to any linked repository and open the `.ipynb` files.
